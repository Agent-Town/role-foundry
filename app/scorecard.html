How to read this scorecard

  • Public curriculum tells you whether the apprentice can learn the visible lessons.
  • Hidden holdouts tell you whether the apprentice can generalize when the exam stays sealed.
  • Compare mode makes deltas explicit so judges can see improvement rather than infer it.
  • Failure analysis shows the curriculum loop: bad hidden-eval results become the next public teaching themes.
